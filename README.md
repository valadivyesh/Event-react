📘 Student Activity Manager — React Project

A complete README with icons, sections, features, and setup steps.

🚀 Project Overview

Student Activity Manager is a fully interactive React-based application designed for students to learn and practice core concepts of modern React development.
This project focuses on strengthening fundamentals like state management, event handling, conditional rendering, refs, props, and component structure—all through a practical, real-use mini application.

The system allows users to add new students, view or hide details, mark their attendance, and delete records.
Its clean UI and logical flow make it perfect for beginners wanting to master React through hands-on experience.

⭐ Key Features
🧩 1. Add New Students

Users can enter student name, age, class, and activity.

All data is stored using React useState.

Validation ensures no empty fields.

👁️ 2. Toggle Student Details

Each student card includes a “Show / Hide Details” control.

Uses conditional rendering to dynamically reveal information.

🧮 3. Attendance System

Students can be marked Present / Absent instantly.

Attendance state updates without refreshing the page.

🗑️ 4. Delete Student

Clean removal of a student entry using array filtering.

Smooth UI feedback and consistent state updates.

💡 5. Clean, Component-Based Architecture

Every UI block is managed by reusable, modular components.

Ensures scalability and readability.

🧵 6. Props & Data Flow

Uses top-down props passing for child components.

Demonstrates how to update parent state from children.

⚙️ Setup & Installation

Follow these steps to run the project locally:

📌 1️⃣ Clone the Repository
git clone https://github.com/your-username/student-activity-manager.git

📌 2️⃣ Navigate to Project Folder
cd student-activity-manager

📌 3️⃣ Install Dependencies
npm install

📌 4️⃣ Start the Development Server
npm start

🌐 After running, visit:
http://localhost:3000
